- Usar o REM para se adapatar as proporções da pagina, se usar o PX ela vai ficar sempre naquele tamanho e vai ser obrigado a usar 
Media Queries

- SPAN - Se usar ele não precisa fazer display flex para ficar na mesma linha

- -webkit-font-smoothing:antialiased; /*Deixa a fonte mais clara*/

- Usar variaveis para padronização de cores:
:root {
  --hue: 159;
  --base-color: hsl(var(--hue) 36% 57%);
  --base-color-second: hsl(var(--hue) 65% 88%);
  --base-color-alt: hsl(var(--hue) 57% 53%);
  --title-color: hsl(var(--hue) 41% 10%);
  --text-color: hsl(0 0% 46%);
  --text-color-light: hsl(0 0% 98%);
  --body-color: hsl(0 0% 98%);
}
Como chamar as variaveis -> background: var(--body-color);

- Padronizar as distancias de uma seção para outra, fazendo com que seja chamado somente 1 vez -> .section {
  padding: 5rem 0;
}

- O Grid faz com que tenha um espaçamento padrão com as classes selecionadas 
.grid {
  display: grid;
  gap: 2rem;
}
